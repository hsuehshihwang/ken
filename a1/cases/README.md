# Cases Directory

This folder contains HTML test players for the A1 English Listening Test.

## Usage

1. Open `test-player.html` in your browser
2. Select a topic from the dropdown
3. Click play buttons to listen to audio

## Adding Your Own Cases

Create new HTML files in this folder for custom test layouts. Each player expects audio files in `../audio/{topic-name}/` with an `index.json` file.
